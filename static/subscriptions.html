Webhook Signature Verification

All webhook deliveries include an X-TriggersAPI-Signature header containing an HMAC-SHA256 signature of the request body. Validate this header on your endpoint to ensure the payload originates from TriggersAPI and has not been tampered with. See API Reference for verification examples.
